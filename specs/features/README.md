# Feature Specs

## How to use this folder
- Before building any new feature, write a spec file here first
- Name it after the feature: e.g. `reddit-buzz-panel.md`, `business-builder.md`
- Each spec should answer:
  1. What is this feature?
  2. Who is it for and what problem does it solve?
  3. Where does it live in the UI?
  4. What data does it need?
  5. What does done look like?
